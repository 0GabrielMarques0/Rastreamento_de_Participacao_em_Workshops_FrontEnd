# Interface WEB - Rastreamento de Participação em Workshops

Este projeto foi gerado usando a versão Angular CLI 19.0.2.

## Servidor de Desenvolvimento

Para iniciar um servidor local de desenvolvimento, execute:

```bash
ng serve
```
Após o servidor ser iniciado, abra seu navegador e acesse http://localhost:4200/. A aplicação será recarregada automaticamente sempre que você modificar algum dos arquivos de origem.
